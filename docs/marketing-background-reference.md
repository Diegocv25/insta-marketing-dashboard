# Referência de background — Insta Marketing

Objetivo: aproximar o fundo do dashboard ao hero da landing principal da Nexus.

## Direção visual assumida
- fundo escuro quase preto
- brilho quente central em marrom/laranja queimado
- sensação de glow suave atrás do conteúdo
- evitar neon frio como base principal do fundo
- manter laranja como cor de destaque/CTA

## Tradução prática
- base: `#0a0507` → `#090509`
- brilho central: marrom queimado / cobre escuro
- acentos: `#ff6a00`

## Uso para agentes
Quando gerar mockups e previews de produto, preferir:
- fundo escuro com glow marrom quente ao centro
- contraste alto com texto claro
- CTA em laranja Nexus
- visual próximo da primeira dobra da landing page
- para peças de **produto**, usar esse fundo marrom desfocado como padrão visual
- para peças de **marca**, manter o visual cyber/neon aprovado
